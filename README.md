# 银河奶牛公会信用点助手

这是一个个人使用的 Tampermonkey 脚本。它只读取公会兑换规则、神龛升级材料和公开市场快照，提供两项只读功能：

- 汇总八种信用点的前五个兑换方案。
- 在游戏原生兑换弹窗选择物品后，按公开收购价估算卖出所得，并计算改买最优兑换物品后可多获得的信用点。
- 信用点卡片可单独折叠，便于快速查看下方颜色的兑换方案。
- 可添加多项神龛升级计划，默认从游戏当前等级开始，合并累计各计划的公会代币与各色公会信用点消耗；也可手动调整起始和目标等级，例如 `0 -> 5` 或 `2 -> 3`。

脚本不会自动购买、出售、兑换或升级。

## 开发命令

```bash
npm test
npm run build
npm run serve
```

## 数据边界

- 市场价格来自公开的 `game_data/marketplace.json` 快照，不发送订单簿或交易请求。
- 兑换规则来自游戏初始化数据；读取失败时会显示错误，不会给出虚假结果。
- 神龛升级材料来自游戏初始化数据中的神龛增益等级表，按每一级实际成本逐项累计。
- 物品名称优先读取当前游戏中文界面物品图标的名称并缓存，因此与已安装的中文界面保持一致；内置词库、究极茶和全等级护符规则仅作为未在页面上渲染过该物品时的兜底。英语和未支持语言保留游戏原名。
- 快照价格不验证订单簿深度，适用于比较兑换材料的估算性价比。
- 买方手续费按 `0` 计算。
- 弹窗的卖出后回购估算使用公开收购价（`b`）和公开卖价（`a`），并从卖出总额中扣除 `2%` 市场税；强化装备会读取当前 `+N` 等级对应的公开收购价，若该等级没有公开收购价则不显示估算。公开快照仍不含订单深度，因此该提示只作比较参考。
- 脚本匹配 `www.milkywayidle.com` 与 `www.milkywayidlecn.com` 的 HTTPS 页面；只有识别到游戏右侧侧栏后才会插入“信用”页签。
