# 银河奶牛公会计算辅助插件

为《银河奶牛放置》（Milky Way Idle）的公会系统提供只读计算辅助，帮助规划公会信用点兑换与神龛升级。

脚本不会自动购买、出售、兑换或升级；不会上传账号数据，也不会读取登录凭据。

## 功能

- 展示八种公会信用点的前五项兑换方案、单点成本与目标成本。
- 支持以市场左一（最低出售价）或右一（最高收购价）作为参考。
- 在游戏原生兑换窗口旁显示当前物品与最优物品的成本对比；原生窗口关闭后提示同步消失。
- 支持多项神龛升级规划，汇总材料、背包库存缺口、预计金币与公会代币成本。
- 可一键把生活或战斗神龛设为当前公会建筑等级对应的目标等级，也可清空全部计划。
- 在材料规划与兑换排行中显示最优兑换物品、所需数量、背包库存；点击物品图标会调用游戏原生的“前往市场”功能并定位该物品。
- 计算公会代币兑换各色信用点的估算金币价值；该区域与八色兑换列表均可折叠。
- 记住目标信用点、价格参考、折叠状态与神龛计划，重新打开浏览器后继续使用。
- 自动适配游戏语言：中文游戏显示完整中文界面；英文或其他非中文游戏语言显示完整英文界面（包含侧栏页签、浮窗、按钮、提示与数字格式）。

## 数据来源与边界

- 市场价格来自游戏公开的 `game_data/marketplace.json` 快照；插件不发送交易或订单簿请求。
- 兑换规则、神龛规则、当前公会等级与角色库存来自当前游戏页面已提供的数据。若关键数据未读取到，插件会明确提示而不是猜测结果。
- 中文物品名称唯一优先来源是游戏官方 i18n 的 `itemNames` 目录；目录暂时不可读时使用上次本地缓存；仍未命中时显示游戏原始英文名。插件不维护静态中文物品表，也不自行拼接护符等名称。
- 快照不含实时订单簿深度；兑换排行与“卖出后回购”提示仅用于估算比较。卖出后回购按公开收购价、公开卖价及 2% 卖方税计算；买方手续费按 0 计算。
- 只有识别到游戏右侧侧栏后，插件才会插入“信用 / Credits”页签；中文与英文原生侧栏均支持。
